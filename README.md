# ToDo_App
This is working ToDo app made with HTML, CSS, Javascript & love, It is also using local storage for better efficiency so that if you somehow closed this you data will be in safe storage.
There is multiple options you can easily add tasks and after completion either you can double-click on task to show it done or delete the task from your ToDo.
Please go through the link and use it for your daily life purposes.

Deployment Link : https://todoappbyanu.netlify.app/
